"Send-Geometry" Remote Volume Rendering

> **This is experimental research code and as such it is prototypical. The code is MIT licensed. **

Build requirements
-------------------------

- [CMake][1] version 2.8 or newer
- [Boost][2]
- OpenGL
- [GLEW][3]
- [GLUT][5] or [FreeGLUT][6]
- [Visionaray][7]
- [Virvo][8]
